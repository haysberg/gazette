Politique de confidentialité

La Gazette Insoumise est un agrégateur de presse statique qui ne collecte aucune donnée personnelle identifiable. Cette page récapitule ce qui se passe (et surtout ce qui ne se passe pas) quand vous nous rendez visite.

Cookies

Aucun cookie n'est déposé par la Gazette. La PWA stocke localement, dans votre navigateur, les ressources statiques nécessaires au fonctionnement hors-ligne (CSS, JS, icônes) via le service worker. Ces données ne quittent jamais votre appareil.

Mesure d'audience

Nous utilisons GoatCounter, un service analytique respectueux de la vie privée : aucun cookie, pas d'empreinte de navigateur, pas d'adresse IP individuelle conservée. Seules sont enregistrées des informations agrégées (page consultée, site référent, pays, catégorie de navigateur). Vous pouvez désactiver le suivi via l'en-tête DNT ou Sec-GPC: 1 (Global Privacy Control), qui sont respectés.

Sites tiers

Les articles agrégés pointent vers des sites indépendants ayant chacun leur propre politique. Cliquer sur un titre vous fait quitter la Gazette : nous ne contrôlons pas ce que ces sites peuvent faire.

Code source

L'intégralité du code est publique sur GitHub. N'importe qui peut vérifier le comportement réel du site.

Contact

Pour signaler un problème de sécurité ou de confidentialité, ouvrez un Security Advisory sur GitHub.
